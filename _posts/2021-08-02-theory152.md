#### 1. 사용자 정의 함수의 개요

- 말 그대로 사용자가 직접 만들어 사용하는 함수이다.

#### 2. C 문제

- 다음은 재귀 함수를 이용해 팩토리얼(Factorial) 연산을 수행하는 C 프로그램이다. 그 실행 결과를 확인하시오.

```c
#include <stdio.h>
int factorial(int n);
main(){
  int (*pf)(int);
  pf = factorial;
  printf("%d", pf(3));
}
int factorial(int n){
  if(n<=1)
    return 1;
  else
    return n*factorial(n-1);
}
```

> 결과: 6
